# 情報フロー（In/Out）

このリポジトリがプロセスする情報の入力と出力。

---

## In（入力）

| 種類 | 内容 |
|------|------|
| **暗号化されたスクリプト** | 無意識の判断・反応・習慣 |
| **体の知恵** | 摂取・消化・排泄に関する経験則 |
| **選択の基準** | マクスウェルの悪魔的に動いているフィルタ条件 |
| **観察** | 「何が効くか」「何が重いか」の気づき |
| **人格・能力の断片** | 言語化されていないアルゴリズムの断片 |

---

## プロセス（変換）

```
In → 客観的特定 → Decrypt → 外部化 → ハードコーディング → Out
```

- **暗号 → 平文**：曖昧なものを明示的な形に変換
- **内部 → 外部**：個人内の処理をリポジトリへ移す
- **動的 → 静的**：実行中の判断をコード・ドキュメントとして固定

---

## Out（出力）

| 種類 | 内容 |
|------|------|
| **ドキュメント** | README、HST理論など、体系化された記述 |
| **ハードコーディングされたルール** | 再現可能な手順・基準 |
| **デジタルツイン** | 人格・能力の再現可能な表現 |
| **バージョン管理された履歴** | 進化の記録 |
| **ストレージ（ブラックホール）** | 情報を吸収する外部記憶 |

---

## 情報の流れ（全体像）

```
[体・脳] 暗号化されたスクリプト
        ↓ In
[リポジトリ] Decrypt → 外部化 → ハードコーディング
        ↓ Out
[ストレージ] 再現可能なアルゴリズム群
        ↓ フィードバック
[体] ホログラム化（軽量化）
```
